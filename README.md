# 🚀 About The Project :-

### API Pariksha is a modern, beginner-friendly API testing platform built for developers, students, and API learners.

It allows users to:
- ⚡ Test REST APIs instantly
- 🛠 Debug API responses easily
- 📡 Send GET, POST, PUT & DELETE requests
- ⏱ Measure response time
- 📑 Manage headers & request body
- 📬 Analyze status codes and response data
- 🧠 Learn API testing visually

# 🔧⚙️ Setup Instructions :-

## 0️⃣ Get the Project Code

### • Clone using Git :-

```bash
git clone https://github.com/reejalchoudhary/api-pariksha-frontend.git
```

```bash
cd api-pariksha-frontend
```

### • OR Download ZIP

Click on **Code → Download ZIP**, then extract and open it in your code editor.

  

1) Install Dependencies:-

 ```
   npm install
   ```
2) Run Development Server:-
 ```
   npm run dev
  ```
3) Open the local dev server link usually at.
```
  http://localhost:5173
```
---

# 🧭 How To Use API Pariksha :-

## 1️⃣ Enter API Endpoint

Paste your API URL inside the request input field.

### Example:

```bash
https://jsonplaceholder.typicode.com/posts
```

---

## 2️⃣ Select HTTP Method

Choose:

- GET
- POST
- PUT
- DELETE

according to your API requirement.

---

## 3️⃣ Add Headers / Params / Body

Configure:

- Request Headers
- Query Parameters
- JSON Request Body

if required.

---

## 4️⃣ Send Request

Click on:

```bash
Send Request 🚀
```

The request will be processed instantly.

---

## 5️⃣ Analyze Response

Inspect:

- ✅ Status Code
- 📦 Response Data
- 📑 Headers
- ⏱ Response Time
